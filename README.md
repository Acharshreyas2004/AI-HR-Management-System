# 🤖 AI HR Management System

An AI-powered HR Management System built using **n8n**, **JavaScript**, **Google Sheets**, **REST APIs**, and **Webhooks**.

---

## 📖 Project Overview

This project automates common Human Resource (HR) operations using workflow automation. It is being developed module by module as part of my MCA project to gain practical experience in backend development, APIs, and automation.

---

# 🛠️ Tech Stack

- n8n
- JavaScript
- Google Sheets
- REST APIs
- Webhooks
- Postman
